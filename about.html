Sobre mim

Olá! Meu nome é Emerson Pinheiro, e sou estudante de Análise e Desenvolvimento de Sistemas. Desde que comecei minha jornada na tecnologia, tenho me dedicado a construir um forte entendimento de várias linguagens de programação e tecnologias essenciais para o desenvolvimento de software. Com um foco em linguagens como C#, JavaScript, HTML, CSS e Python, tenho ampliado minhas habilidades para me tornar um desenvolvedor full stack, capaz de trabalhar tanto no front-end quanto no back-end de aplicações. Além disso, adquira conhecimento em frameworks e ferramentas essenciais, como .NET, para desenvolvimento eficiente de aplicações robustas e escaláveis. Meu conhecimento em bancos de dados SQL me permite trabalhar com o armazenamento e a manipulação de dados, oferecendo uma visão mais ampla sobre a importância da organização de dados na tomada de decisões e no desenvolvimento de sistemas de qualidade. Estou sempre buscando aprender e explorar novas tecnologias para me manter atualizado com as melhores práticas do mercado e oferecer soluções inovadoras e eficientes para problemas complexos. No momento, estou aprimorando minhas habilidades em arquitetura de sistemas e metodologias para garantir que cada projeto em que eu trabalho seja bem estruturado e alinhado com as necessidades do cliente.

Ferramentas Usadas: Para manter um fluxo de trabalho eficiente e bem organizado, utlizo ferramentas que cobrem cada etapa do desenvolvimento de sistemas. No GitHub, faço o controle de versão dos meus projetos, garantindo a rastreabilidade das mudanças e a colaboração com outros desenvolvedores. Como editores principais, uso o Visual Studio Code para desenvolvimento front-end e o Visual Studio para projetos em .NET e C#, ambos essenciais para aumentar minha produtividade e qualidade no desenvolvimento. O Node.js é outra ferramneta fundamental,permitindo que eu execute JavaScript no servidor e execute aplicativos web rápidos e escaláveis. Para o gerenciamento de tarefas, ultilizo o Trello, onde organizo as atividades de cada projeto, aplicativo a etodologia Ágil Scrum. Essa metodologia me ajuda a dividir o trabalho em cilcos,priorizar entregas e garantir uma comunicação clara e eficaz em cada etapa de desenvolvimento
